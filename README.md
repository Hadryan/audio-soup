# audio-soup
Interactive Machine Learning tool designed for sample review and feature selection on audio datasets.

## quickstart
```
env FLASK_APP=appy.py flask run
```

## loading dataset

```
python manage.py load_dataset 'dataset_dir' 'dataset_name' 'dataset_url'
```
